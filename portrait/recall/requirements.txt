# This layer is OPTIONAL. Without these, recall.py fails soft and tells you
# to install them; nothing else in Portrait depends on it. grep is the
# always-available, exact, transparent fallback — this only ADDS fuzzy recall.
#
#   pip install -r recall/requirements.txt
#
# NOTE — modern Python (PEP 668 "externally-managed-environment", common on
# recent macOS/Homebrew/Debian) blocks that system-wide. Use a venv (recommended):
#   python3 -m venv .venv && .venv/bin/pip install -r recall/requirements.txt
# or pipx, or (accepting the risk) pip install --break-system-packages -r recall/requirements.txt
#

# local ONNX runtime + downloads the BAAI/bge-small-en-v1.5 model weights
# (~130 MB) on first run, cached locally under recall/.model_cache/. No torch,
# no server, no daemon, no Docker — but the embedding model IS a real dependency.
