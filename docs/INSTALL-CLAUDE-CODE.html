First-time setup

Get Claude Code

Never used Claude Code — or the Terminal — before? Start here. Every command has a Copy button: paste it into a Terminal window, press return, wait for it to finish, then do the next one.

For macOS. Works offline once the page has loaded (the brand fonts fetch when you're online). On Windows or Linux? See the box at the very bottom.

You are here — this page gets Claude Code itself working on your Mac. Then:
1 · this page  →  2 · start.html — install svaha and run your first loop

One thing to check first — you need a paid Claude plan

Claude Code needs a Claude Pro, Max, Team, or Enterprise subscription, or a Claude Console (pay-as-you-go API) account. The free Claude.ai plan does not include Claude Code — if you only have the free plan, the login step at the end will turn you away. You also need macOS 13 or newer. Everything else below installs itself.

Terminal = a command — paste it & press return.   On screen = do this with the mouse (a menu, a button, a dialog).   Claude = type into a running Claude Code session.

0 · Open the Terminal

0On screen  The Terminal is the app where you paste these commands. To open it:

Press ⌘ Cmd + Space to open Spotlight search, type Terminal, and press return. A small window opens with a blinking cursor.

You'll see a line ending in % (or $), like: yourname@MacBook ~ % That's the prompt. It's waiting for you. Leave this window open for the whole guide.

Brand new to the Terminal?

Nothing here can harm your Mac — you're only ever pasting one short, known command at a time and watching what it prints. If a command seems to "hang," it's usually still working; give it a moment before touching anything. Anthropic also has a gentle terminal guide if you'd like the longer walkthrough.

1 · Install Claude Code

1Terminal  Paste this one line and press return. It downloads and installs Claude Code — the recommended way, and it brings everything it needs with it (you do not need to install Node.js or anything else first):
curl -fsSL https://claude.ai/install.sh | bash
It prints a few lines of progress and ends with a success message naming the installed version. From now on it updates itself quietly in the background — you won't have to reinstall.

Prefer Homebrew or npm? (optional)

The line above is the simplest path. If you already live in one of these, they work too:

brew install --cask claude-code

…or via npm (this one needs Node.js 18+ installed already — the native installer above does not):

npm install -g @anthropic-ai/claude-code

2 · Check it installed

2Terminal  Confirm the claude command now exists:
claude --version
prints a version number, e.g. 2.1.x (Claude Code). If instead you see command not found, quit Terminal and reopen it (the installer added claude to your path, and a fresh window picks it up), then try again. Still stuck? See If something snags.

3 · Log in

3Terminal  Start Claude Code for the first time:
claude
On screen  A browser window opens. Sign in with your Claude account (the paid plan from the box at the top), approve the request, then come back to the Terminal — it will say you're connected.
After this, your login is remembered — you won't have to do it again. (To switch accounts later, type /login inside a session.)
Claude  You're now inside Claude Code. To step back out to the Terminal for the next steps, type:
/exit

4 · Health check (optional, recommended)

4Terminal  Claude Code ships a built-in doctor that checks its own install and config:
claude doctor
walks down a checklist — install, version, auth, paths — ideally all ✓. It tells you exactly what to fix if anything's off.

5 · Two more things, so svaha installs cleanly

svaha's install (start.html, your next stop) uses three small tools that don't ship with a fresh Mac: git (to download svaha), python3 (its safety hooks are Python), and jq (its six shell hooks read their input with jq and silently do nothing without it). One Apple command installs the first two; brew installs jq.

5Terminal  Ask macOS to install the Command Line Tools:
xcode-select --install
On screen  A dialog pops up — click Install and let it finish (a few minutes). If it instead says "already installed," you're already set — move on.
Terminal  Then install jq with Homebrew:
brew install jq
Terminal  Confirm all three tools are now there:
git --version && python3 --version && jq --version
three version lines, for git, Python 3, and jq. If all three print, every check on start.html's first step will pass.

✓ You're ready

Claude Code is installed, logged in, and the three helper tools are in place. Next: open start.html (in this same folder) and follow it to install svaha on top. Its very first step — the claude --version && python3 --version && git --version && jq --version check — should now come up all green.

If something snags

You see…Do this
claude: command not found right after installingQuit Terminal completely (⌘ Cmd+Q) and reopen it, then retry claude --version. A new window reloads your path. If it still fails, run the installer line again.
The login browser rejects you / says no Claude Code accessYou're on the free plan. Claude Code needs Pro, Max, Team, Enterprise, or a Console (API) account — upgrade at claude.com/pricing, then run claude again.
Spotlight doesn't find "Terminal"Open Finder → Applications → Utilities → Terminal, or in Spotlight type the full word Terminal.app.
The command pasted oddly / "command not found" on a line you copiedUse the Copy button (it grabs the exact text) rather than selecting by hand, which can pick up stray characters. Paste one command at a time, never a block.
curl: command not foundRun xcode-select --install first (Step 5) — it provides curl too — then retry Step 1.
Anything else during install or loginClaude's own install & login troubleshooting covers the rest, and claude doctor (Step 4) names most problems for you.

On Windows or Linux instead?

This guide is written for macOS. The svaha steps that follow are the same everywhere, but getting Claude Code itself differs slightly:

Windows (PowerShell):

irm https://claude.ai/install.ps1 | iex

Linux / WSL: same line as macOS — curl -fsSL https://claude.ai/install.sh | bash.

Full platform-by-platform instructions: code.claude.com/docs/en/setup. On Windows, install Git for Windows (covers Step 5's git), Python 3, and jq separately.
