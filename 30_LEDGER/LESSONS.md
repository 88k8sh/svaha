# LESSONS — Named Lessons from Incidents

Append-only. Distilled, human-readable lessons — each derived from a real incident (see `session-fixes.md` for the raw bug log).
A lesson earns a name when the same class of mistake could recur; the name makes it citable in future sessions.

---

## Format

```
## [Lesson Name]
- **Incident:** [what happened, one line]
- **Lesson:** [the generalizable rule]
- **Guard:** [the mechanism that now prevents recurrence, if any]
```
